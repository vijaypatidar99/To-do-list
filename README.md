# Todo Application
Todo application developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add todo
- Edit todo
- List Dodo
- Delete Todo
- Login
- Signup
- Responsive layout

## Technology stack
- Java 8+
- JSP and Servlet
- JDBC
- MySQL database
- Eclipse IDE
- Apache Tomcat server 8+

The name of our database is todo_management.
It consists of two tables namely users and todos.
The users table contains the data of registered users.
Whereas the todos table contains all the todos entered by users.
